# たんぽぽ書道教室

書道教室の先生と保護者を繋ぐ連絡用webアプリ
![image](https://user-images.githubusercontent.com/106829132/232237535-2becef9d-f075-4d9e-886d-f44a74ab6f40.png)

運営者（先生）から生徒の状況や、教室での活動内容を効率的に保護者と共有できるシステムが必要との要望があり開発を行った。
